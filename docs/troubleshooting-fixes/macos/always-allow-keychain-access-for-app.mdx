<img
  src={KeyChainPromptScreenshot}

  alt="macOS Keychain Prompt"
/>

1. Open Keychain Access in **Applications > Utilities > Keychain Access**
2. Highlight **"Login"** Keychain
   <img
     src={KeyChainScreenshot}
     style={{ height: "25vh" }}
     alt="macOS Keychain"
   />
3. Find the key or keys you wish to allow your application to access
4. Right-click the key item, and select **"Get Info"**
5. Click on the **"Access Control"** tab
6. Select **"Allow all application to access this item"** or **"Always Allow"**

   <Admonition type="caution" icon="⚠️" title="Carefull with this option ...">
     <p>
       If you select <b>"Allow all application to access this item"</b>, this
       will allow all applications to access the keychain item.
     </p>
   </Admonition>

7. Press **"Save Changes"** to confirm the change
8. Enter your password you use to login to the server to authenticate.
